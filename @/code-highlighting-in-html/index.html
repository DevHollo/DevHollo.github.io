Code Highlighting In HTML

How to use:

Just put this in your head tag:


<link rel="stylesheet" type="text/css" href="https://devhollo.github.io/@/code-highlighting-in-html/css/codemirror.css">
<script src="https://devhollo.github.io/@/code-highlighting-in-html/js/codemirror.js"></script>
<script src="https://devhollo.github.io/@/code-highlighting-in-html/js/langs/LANGUAGE_NAME.js"></script>
<script src="https://devhollo.github.io/@/code-highlighting-in-html/js/loaders/LANGUAGE_NAME.js" defer></script>

lllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllllll
